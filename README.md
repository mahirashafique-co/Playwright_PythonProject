## How To Run Tests

### Install Dependencies
pip install playwright
pip install pytest-playwright
pip install pytest-xdist
playwright install

### Run Single Test
pytest tests/test_google.py --headed

### Run All Tests
pytest --headed

### Run On All Browsers
pytest --browser chromium --browser firefox --browser webkit
